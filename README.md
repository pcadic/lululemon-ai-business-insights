## üìå Project Overview

**Lululemon AI Business Insights** collects **real customer reviews** from Lululemon stores in Vancouver and transforms them into **business KPIs and insights** using **Hugging Face NLP models**.  

All data processing is **automated via GitHub Actions**, and results are presented in a **cloud-based Streamlit dashboard**.

- **Storage:** CSV files versioned in GitHub
